#Air Traffic

#Instalation
1. Install dependencies: composer install
2. Run the database 
3. Add .env file from .env.example
4. Run serve: php -S localhost:8000

#Run test
1. Install dependencies: composer install
2. Run the database 
3. Add .env file from .env.example
4. Run serve: php -S localhost:8000
3. Run test: php vendor/phpunit/phpunit/phpunit test
